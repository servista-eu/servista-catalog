**Push mirror creation (per D-03, if user chose provide-pat in Task 1):**

Extract Forgejo token:
```bash
TOKEN=$(git remote get-url origin | sed -n 's|https://heaphopdancer:\([^@]*\)@.*|\1|p')
```

Create new push mirror using the GitHub username and PAT provided by the user:
```bash
curl -s -X POST \
  -H "Authorization: token $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "remote_address": "https://github.com/servista-eu/qabatz-catalog.git",
    "remote_username": "<GITHUB_USERNAME_FROM_TASK1>",
    "remote_password": "<GITHUB_PAT_FROM_TASK1>",
    "interval": "8h0m0s",
    "sync_on_commit": true
  }' \
  "https://git.hestia-ng.eu/api/v1/repos/qabatz/qabatz-catalog/push_mirrors"
```

```bash
curl -s -H "Authorization: token $TOKEN" \
```
